1. copy  Tab32x20.ocx and tabpro20.lic files into windows system32 directory in the target 	machine

2. register Tab32x20.ocx using regsvr32 exe.

 usage : regsvr32 <file name> in Start->Run command.
 
After registering, the messagebox will be displayed with the message
 "DllRegisterServer in <file name> succeeded"
